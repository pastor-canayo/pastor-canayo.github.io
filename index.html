ACH Review

Activating a card

Activating a card => System Administration => change card status => enter card number and then select the card status to active.

Checking funds from a canceled check

Customer Account => account info => adjustment history to see when the check was issued. Then go to pragram refund account => bill payments => enter the dates a day before and the date of the check. grab the paymentID and paste on the CH account notes to compare => then go to all transactions to see if the payment is there. If payment is back on the refund account it will say BILL PAY REFUND on the description.

Trasfering funds

Refund account => finance bar => insert adjustment => enter the negative amount => type "v" => then type in comments: transferring funds from a canceled check to CH. CH account => finance bar => insert adjustment => enter the positive amount => type "v" => then type in comments: funds returning from refund account to CH account from canceled check.

Post a DD marked for return

System Administration => other tools => pending ACH returns => chose super all => then paste transID => click update and DD will show => here chose "E" for return => then go to DD clearing and find the return and the click in process in change => and you are done.

misceleaneus

When you see pending transaction in all transactions tab, you can go to misceleaneus to remove the hold days by changing it to "zero" => then click update and the amount posted should show up on the balance.

Refunds time frame

=> Canceled check takes from 24-36 hours for refunds to return.
=> Expedited check takes 2 business day.
=> When refund check is sent out standard mail it will take from 7 to 14 business days.

Driving an account negative when NSF for returning

First add the total how much you need for the account balance for example if there is only 40.00 on the account, and then you have to reuturn 50.00 => you go to finance bar => transfer funds => add 10.00 with Q type adjustment => it will show in the balance the 50.00 => payment history => return the 50.00 => after that go to finance bar again => to substract the -10.00 with Q type adjustment and this will show the -10 on the account balance.
